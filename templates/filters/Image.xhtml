The "image" filter

Goal : turns a text entry field into an image URL entry field to include images by URL within a document.

The "image" filter is compatible with the "text" plugin. It doesn't upload any image file on the server, it simply displays the existing file inside an img tag.

The examples below are shown inside a xt:repeat element so that you can see how does look a freshly repeated element.

The example below set an "image" class attribute on the image container and on the entry field. The border and margins are set with CSS rules applied to a div.image, input.image selector.
